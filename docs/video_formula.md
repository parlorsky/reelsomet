# Формула видео для Reels

## Структура видео по секундам

```
┌─────────────────────────────────────────────────────────────────┐
│  0-3с    │  HOOK VIDEO (исходное качество)                      │
│          │  Видео-крючок без субтитров, привлекает внимание     │
├──────────┼──────────────────────────────────────────────────────┤
│  3-6с    │  FREEZE FRAME + INTRO TEXT                           │
│          │  Последний кадр хука + текст вступления:             │
│          │  "Да похуй что {контекст хука}                       │
│          │   Давайте {ситуативно}"                              │
├──────────┼──────────────────────────────────────────────────────┤
│  6-12с   │  AUTHOR QUOTE                                        │
│          │  Имя автора + название книги + ключевая цитата       │
│          │  Фон: динамический из каталога                       │
├──────────┼──────────────────────────────────────────────────────┤
│  12-45с  │  MAIN CONTENT                                        │
│          │  Основной контент с субтитрами                       │
│          │  Смена фона на каждой странице (---)                 │
├──────────┼──────────────────────────────────────────────────────┤
│  45-55с  │  CALL TO ACTION                                      │
│          │  Призыв к действию, финальная мысль                  │
├──────────┼──────────────────────────────────────────────────────┤
│  55-60с  │  OUTRO / BRANDING                                    │
│          │  Логотип, подписка, ссылка                           │
└──────────┴──────────────────────────────────────────────────────┘
```

## Формула вступления

```
"Да похуй что {X}. Давайте {Y}."

Где:
- {X} = контекст из хука (что говорят/думают люди)
- {Y} = призыв к размышлению/действию
```

### Примеры вступлений:

| Тема | {X} | {Y} |
|------|-----|-----|
| Любовь | он там думает | лучше разберёмся |
| Продуктивность | все говорят про успех | задумаемся о главном |
| Отношения | было раньше | лучше вспомним |
| Деньги | тебе советуют | посчитаем сами |
| Здоровье | пишут в интернете | послушаем науку |

## Формат скрипта с хуком

```
@hook: downloads/hooks/motivation_hook.mp4
@hook_duration: 3.0
@freeze_duration: 3.0

---

_«Да_ _похуй_ _что_ _он_ _там_ _думает_
**Давайте** лучше **разберёмся»**

---

[c:cyan]Эрих[/] [c:cyan]Фром[/] _в_ _книге_
**«Искусство** **любить»** пишет:

---

... основной контент ...
```

## Параметры хука

| Параметр | Описание | По умолчанию |
|----------|----------|--------------|
| `@hook` | Путь к видео-хуку | - |
| `@hook_duration` | Длительность показа хука (сек) | 3.0 |
| `@freeze_duration` | Длительность заморозки (сек) | 3.0 |
| `@hook_volume` | Громкость звука хука (0-1) | 0.0 |

## Требования к хук-видео

1. **Формат**: MP4, вертикальный (9:16)
2. **Разрешение**: 1080x1920 или выше
3. **Длительность**: 2-5 секунд
4. **Контент**:
   - Яркий, привлекающий внимание
   - Без текста (субтитры добавятся)
   - Эмоциональный момент

## Папка для хуков

```
downloads/
└── hooks/
    ├── motivation_01.mp4
    ├── love_01.mp4
    ├── money_01.mp4
    └── catalog.json  (метаданные хуков)
```

## Использование

```bash
# С хуком
python scripts/styled_subtitles.py script_with_hook.txt audio.mp3 \
    --hook downloads/hooks/love_01.mp4 \
    --hook-duration 3 \
    --freeze-duration 3 \
    -o output.mp4

# Без хука (как раньше)
python scripts/styled_subtitles.py script.txt audio.mp3 -o output.mp4
```
